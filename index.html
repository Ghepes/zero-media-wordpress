ZERO MEDIA WordPress

Implement a comprehensive solution for replacing all media types in WordPress with external media URLs, fully supporting popular page builders and plugins such as Elementor, WPBakery Page Builder, and Slider Revolution. This innovative approach transforms your WordPress site into a zero-media webpage, leveraging external CDN links to seamlessly integrate high-quality media content. Enhance your business website by embedding the most impressive media via URL links, optimizing performance and resource management.

Overview Installation

Install it the same as a general website, but you don't need to install MySQL/database.

  1. Download ZERO MEDIA WordPress from codecanyon.net
  2. Extract/Unzip ZERO MEDIA WordPress Files and copy/upload ZERO MEDIA WordPress to your server
  3. By FTP must be added in wp-content folder
  4. Then activate zero-media-wordpress.
  5. done!

Installation Instructions

STEP - Download ZERO MEDIA WordPress

You can download that file by logging into https://codecanyon.net/downloads and clicking the Downloads Tab

  1. The first step is to download Zero Media WP from the codecanyon website (https://codecanyon.net/downloads). You need two files, namely:
    • All files end documentation (codecanyon-xxx.zip)
    • License certificate end purchase code (text) (xxx-license.txt)
  2. Then extract the zip file (codecanyon-xxx.zip), you will get two zip files, namely: Documents (documents.zip) and Zero Media WP installation file (rev2x.xx.xx.zip).

STEP - Extract/Unzip ZERO MEDIA WordPress Files from Envato

Please extract your ZERO MEDIA WordPress file (re1.xx.xx.zip) using WinRAR/WinZip/7Zip/Tar and copy it into your folder. The main plugin folder ready for installation is in the folder named Zip_Plugins_to_add_to_WordPress/zero-media-wordpress.zip.

  • All files & documentation (codecanyon-xxx.zip)
  • License certificate & purchase code (text) (xxx-license.txt)

STEP - Upload via FTP

  • The step to upload via FTP ZERO MEDIA WordPress for cPanel. Then extract the zip file (codecanyon-xxx.zip), select /zero-media-wordpress.zip and upload via FTP.
    • All files & documentation (codecanyon-xxx.zip)
    • License certificate & purchase code (text) (xxx-license.txt)
  • The next step is to log in to your web hosting cPanel account.
  • Once you are logged in, navigate to File Section and open the File Manager.

    • STEP - Folder public_html FTP

  • Open the public_html folder and click on the Upload option on the cPanel menu bar.

    • STEP - Install through the WordPress Plugins panel

    Login to your WordPress panel, upload the plugin zip file to WordPress, install, and activate the plugin: https://wordpress.org/download/

    STEP - Envato Purchase Code License

    STEP - Envato Purchase Code License 'license.txt' or 'license.pdf' File

  • On Setup configuration tab you are asked to enter Your Purchase code and Email address, you can find it in the xxxxx-license.txt file

    • STEP - Activate License to wordpress

    Open the WordPress admin panel, click the "Zero Media WP License". Copy the purchased Envato code from 'license.txt' File and add it to the "Zero Media WP License" in Wordpress plugins section, click the button to activate License. Plugin is activated, if the license is correct or correctly copied.

    STEP - Media Library new sections

    Open the Media Library panel, click on a media file. You will see 3 new sections on media files:

    • 1 - Generate SVG.
    • 2 - Externals Media URL.
    • 3 - Save External Media URL.

    STEP - 1. Generate SVG

    In section 1: Click the "Generate SVG" button, then go back to the main media.

    I purchased ZERO MEDIA WordPress, should I purchase ZERO MEDIA WordPress again after 1 year?

    If you have already purchased ZERO MEDIA WordPress, you do not need to purchase it again. You can use the same license without support and you can use an older version.

    What is the difference between versions ?

    The main difference between versions of the plugin lies in the features and compatibility with different WordPress versions. Make sure to check the changelog for detailed information.

    How to export plugin after the license expires?

    To export the plugin after the license expires, you can use the export tools built into your profile, but with an expired license you will no longer be able to receive new versions or the support granted during the license.

    What types of Media does the Plugin with External links support?

    The plugin supports a wide range of media types including images, videos, audio files, and documents. Refer to the documentation for a complete list of supported media types.

    What is the period of the License?

    The period is one year, from the day you activated it.

    How long do I receive support?

    With each Activation there is a period of 6 Months of support. And you can extend the support period.

    Is there a video tutorial on how to use Zero Media Wordpress plugins?

    Yes. There is a wide range of video media where you can find out about Zero Media Wordpress. And from time to time we add new videos, to help you better discover this plugin and how it can help you improve your website performance
